Glossaire AutoCAD

dwg
format de fichier par défaut d’AutoCAD. le format de fichier dwg change tous les 3 ans. Ainsi, avec Autocad 2007, on peut ouvrir tous les formats dwg des versions antérieures, mais aussi (fait rare ...) les fichiers acad 2007 + 2008 + 2009. Avec la version AutoCAD 2010, un nouveau format dwg apparaît. On pourra ouvrir les fichiers dwg jusqu’à AutoCAD 2013, ainsi de suite ...
