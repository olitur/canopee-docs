Glossaire Bureautique
====================== 

.. glossary::
  :sorted:

  pdf
    Portable Document Format : format de fichier de **visualisation** : permet d'avoir une représentation fidèle de ce qui peut être imprimé. Utilisé partout et par tous.

    C'est aussi un mode de **sauvegarde**, car les imrimantes *virtuelles* qui génèrent ces type de document le font de plus en plus en mode *vectoriel*
